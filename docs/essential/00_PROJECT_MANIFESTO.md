# Projeto: ness. (n.process) - Middleware de Inteligência

## 1. Visão do Produto
O **n.process** não é um aplicativo final para usuário comum. É uma **Plataforma de Infraestrutura (Control Plane)** que fornece capacidades de IA (BPMN, Compliance, Docs) para outros sistemas (ERPs, CRMs) via API e MCP.
O produto é "powered by **ness.**".

## 2. Branding & Identity
- **Nome do Produto:** `n.process` (Sempre minúsculo).
- **Logotipo:** Fonte **Montserrat Medium**. Cor do texto: Preto ou Branco (contraste).
- **O Ponto:** O caractere `.` deve ser **Ciano (#00ade8)**.
- **Estilo Visual:** "Industrial Tech", "Dark Mode Profundo", "Bento Grids". Foco em dados e status, não em enfeites.

## 3. Os 4 Motores (Core Value)
1.  **Process Engine:** Gera diagramas BPMN 2.0 a partir de texto/áudio.
2.  **Compliance Guard:** Audita processos contra leis (RAG).
3.  **Document Factory:** Gera PDFs/Manuais oficiais.
4.  **Knowledge Store:** O "cérebro" (RAG + MCP) compartilhado ou privado.
